# 浮点数

实践是检验真理的唯一标准

## float

占 $32bit = 4B$, 在不考虑精度的情况下能定性地表示 $10^{38}$ 的大小, 在精度为 $1$ 的情况下能表示 $10^{10}$ 以内的数字

## double

占 $64bit = 8B$, 在不考虑精度的情况下能定性地表示 $10^{308}$ 的大小, 在精度为 $1$ 的情况下能表示 $10^{22}$ 以内的数字

## long double

占 $128bit = 16B$, 在不考虑精度的情况下能定性地表示 $10^{4932}$ 的大小, 在精度为 $1$ 的情况下能表示 $10^{27}$ 以内的数字
