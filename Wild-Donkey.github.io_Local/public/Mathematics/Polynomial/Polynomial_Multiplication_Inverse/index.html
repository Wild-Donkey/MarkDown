多项式乘法逆

多项式乘法逆

定义

对于 n−1n - 1n−1 次的多项式 fff, 找出一个 ggg 使得 f×g≡1(modxn)f \times g \equiv 1 \pmod {x^n}f×g≡1(modxn).

理论

当 n=0n = 0n=0 的时候, 显然 g0=1f0g_0 = \frac 1{f_0}g0​=f0​1​.

现在对 n>0n > 0n>0 的情况设有已知的 g′g'g′ 使得 f×g′≡1(modx⌈n2⌉)f \times g' \equiv 1 \pmod {x^{\lceil \frac n2\rceil}}f×g′≡1(modx⌈2n​⌉), 则有:

f×g≡1(modxn)f×g≡1(modx⌈n2⌉)f×g′≡1(modx⌈n2⌉)f×(g−g′)≡0(modx⌈n2⌉)\begin{aligned} f \times g &\equiv 1 \pmod {x^n}\\ f \times g &\equiv 1 \pmod {x^{\lceil \frac n2\rceil}}\\ f \times g' &\equiv 1 \pmod {x^{\lceil \frac n2\rceil}}\\ f \times (g - g') &\equiv 0 \pmod {x^{\lceil \frac n2\rceil}} \end{aligned} f×gf×gf×g′f×(g−g′)​≡1(modxn)≡1(modx⌈2n​⌉)≡1(modx⌈2n​⌉)≡0(modx⌈2n​⌉)​

我们知道, 如果 f0=0f_0 = 0f0​=0, 那么 ggg 就不存在, 因为无法凑出 f×gf \times gf×g 的零次项. 因此 g−g′≡0(modx⌈n2⌉)g - g' \equiv 0 \pmod {x^{\lceil \frac n2\rceil}}g−g′≡0(modx⌈2n​⌉).

接下来通过平方, 把模 x⌈n2⌉x^{\lceil \frac n2\rceil}x⌈2n​⌉ 意义下转化成模 xnx^nxn 意义下:

g−g′≡0(modx⌈n2⌉)(g−g′)2≡0(modxn)g2−2g′g+g′2≡0(modxn)\begin{aligned} g - g' &\equiv 0 \pmod {x^{\lceil \frac n2\rceil}}\\ (g - g')^2 &\equiv 0 \pmod {x^n}\\ g^2 - 2g'g + {g'}^2 &\equiv 0 \pmod {x^n}\\ \end{aligned} g−g′(g−g′)2g2−2g′g+g′2​≡0(modx⌈2n​⌉)≡0(modxn)≡0(modxn)​

两边同时乘以 fff:

g−2g′+fg′2≡0(modxn)g≡2g′−fg′2(modxn)\begin{aligned} g - 2g' + f{g'}^2 &\equiv 0 \pmod {x^n}\\ g &\equiv 2g' - f{g'}^2 \pmod {x^n}\\ \end{aligned} g−2g′+fg′2g​≡0(modxn)≡2g′−fg′2(modxn)​

通过 FFT 进行两次乘法操作然后加减可以求出 ggg, 我们需要递归求解, 一共是 log⁡n\log nlogn 层, 每层复杂度是 O(nlog⁡n)O(n\log n)O(nlogn), 总复杂度是 T(n)=T(n2)+O(nlog⁡n)T(n) = T(\frac n2) + O(n \log n)T(n)=T(2n​)+O(nlogn), 根据主定理, T(n)=O(nlog⁡n)T(n) = O(n \log n)T(n)=O(nlogn).

实现

仍然是将多项式乘法浅封装一下, 然后直接递归即可.
