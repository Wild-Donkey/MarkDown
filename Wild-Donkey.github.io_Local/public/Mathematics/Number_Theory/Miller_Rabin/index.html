Miller-Rabin 素性检验

Miller-Rabin 素性检验

费马素性检验

费马小定理提出, 也就是对于所有小于 nnn 的正整数 aaa, 有 an−1≡1(modn)a^{n - 1} \equiv 1 \pmod nan−1≡1(modn).

所以只要我们可以找到一个 a<na < na<n, 使得 an−1≢1(modn)a^{n - 1} \not\equiv 1 \pmod nan−1​≡1(modn), 那么就可以确定 nnn 不是一个素数.

对于合数 nnn, 一定存在 aaa 使得 an−1≢1(modn)a^{n - 1} \not\equiv 1 \pmod nan−1​≡1(modn). 取 aaa 使得 gcd⁡(a,b)≠1\gcd(a, b) \neq 1gcd(a,b)​=1, 会发现无论 aaa 的多少次方, 都不会出现对 gcd⁡(a,b)\gcd(a, b)gcd(a,b) 取模不为 000 的结果, 所以一定不会出现 111.

我们随机找一个正整数 a<na < na<n, 测试 an−1a^{n - 1}an−1 对 nnn 取模的结果是否为 111. 重复此过程一定的次数, 如果都是 111, 即可在一定的概率下得到 nnn 是一个质数的结论. 否则就可以断定 nnn 是合数. 这就是费马素性检验.

值得注意的是, 对于 gcd⁡(a,n)=1\gcd(a, n) = 1gcd(a,n)=1 的 aaa, 没有定理证明了一定存在这样的 aaa 使得 an−1≡1(modn)a^{n - 1} \equiv 1 \pmod nan−1≡1(modn). 事实上, 可以找到这样的反例, 当 n=561n = 561n=561 时, 对于所有 gcd⁡(a,n)=1\gcd(a, n) = 1gcd(a,n)=1 的 aaa, 都有 an−1≡1(modn)a^{n - 1} \equiv 1 \pmod nan−1≡1(modn). 这样的反例可以找到很多, 比如 10510^5105 以内的反例就有: 561561561, 110511051105, 172917291729, 246524652465, 282128212821, 660166016601, 891189118911, 105851058510585, 158411584115841, 293412934129341, 410414104141041, 466574665746657, 526335263352633, 627456274562745, 639736397363973, 753617536175361.

这种数叫做卡迈克尔数, 在 OEIS 中有数列A002997介绍了这种数. 费马素性检验在遇到卡迈克尔数的时候的正确率非常差, 因此一般不用于实践.

模意义下 111 的平方根

对于模 nnn 意义下 111 的平方根 xxx, 满足 x2≡1(modn)x^2 \equiv 1 \pmod nx2≡1(modn).

x2≡1(modn)x2−1≡0(modn)(x−1)(x+1)≡0(modn)kn=(x−1)(x+1)\begin{aligned} x^2 &\equiv 1 \pmod n\\ x^2 - 1 &\equiv 0 \pmod n\\ (x - 1)(x + 1) &\equiv 0 \pmod n\\ kn &= (x - 1)(x + 1)\\ \end{aligned} x2x2−1(x−1)(x+1)kn​≡1(modn)≡0(modn)≡0(modn)=(x−1)(x+1)​

如果 nnn 是一个质数, 那么显然 xxx 只有两种情况 111 或 n−1n - 1n−1. 因为本来不含因子 nnn 的两个正整数 aaa, bbb 相乘后仍然不含因子 nnn, 所以模 nnn 意义下 x−1x - 1x−1 和 x+1x + 1x+1 必有一个为 000.

但是如果 nnn 是合数, xxx 就可以是别的数了, 因此如果模 nnn 意义下一个数 xxx 的平方为 111, 而 xxx 不是 111 或 −1-1−1, 那么 nnn 无疑是一个合数. Miller-Rabin 素性检验就是在费马素性检验的基础上多判了这种情况.

Miller-Rabin 素性检验

我们把 n−1n - 1n−1 表示成 2s2^s2s 和奇数 ddd 的乘积, 重新审视费马小定理的式子:

an−1≡1(modn)a2sd≡1(modn)a2sd−1≡0(modn)(a2s−1d−1)(a2s−1d+1)≡0(modn)(ad−1)∏i=0s−1(a2id+1)≡0(modn)\begin{aligned} a^{n - 1} &\equiv 1 \pmod n\\ a^{2^sd} &\equiv 1 \pmod n\\ a^{2^sd} - 1 &\equiv 0 \pmod n\\ (a^{2^{s - 1}d} - 1)(a^{2^{s - 1}d} + 1) &\equiv 0 \pmod n\\ (a^d - 1)\prod_{i = 0}^{s - 1}(a^{2^id} + 1) &\equiv 0 \pmod n\\ \end{aligned} an−1a2sda2sd−1(a2s−1d−1)(a2s−1d+1)(ad−1)i=0∏s−1​(a2id+1)​≡1(modn)≡1(modn)≡0(modn)≡0(modn)≡0(modn)​

所以 ad−1a^d - 1ad−1, a2id+1a^{2^id} + 1a2id+1 在模 nnn 意义下只要有一个是 000 即可认为 an−1≡1(modn)a^{n - 1} \equiv 1 \pmod nan−1≡1(modn). 这样就不会放过费马素性检验所能检验的所有情况.

nnn 如果是合数的话, 可能存在所有 ad−1a^d - 1ad−1, a2id+1a^{2^id} + 1a2id+1 都不是 000, 但是满足 an−1≡1(modn)a^{n - 1} \equiv 1 \pmod nan−1≡1(modn), 这是因为这些非零项相乘时得到了 nnn 的倍数, 所以得到最终结果为 000. 这种情况不会被费马素性检验判为合数, 但是 Miller-Rabin 却可以判为合数.

我们思考这样判断的理由.

如果遇到 a2i−1da^{2^{i - 1}d}a2i−1d 不是 111 也不是 −1-1−1, 但是 a2ida^{2^id}a2id 却是 111, 根据前面对 111 的平方根的研究, 此时 nnn 一定是个合数.

我们把 ad−1a^d - 1ad−1, ad+1a^d + 1ad+1, a2d+1a^{2d} + 1a2d+1,…, a2s−1d+1a^{2^{s - 1}d} + 1a2s−1d+1 排成一个序列.

我们发现序列中最多只会出现一个 000, 有 000 则必定 an−1≡1(modn)a^{n - 1} \equiv 1 \pmod nan−1≡1(modn). 讨论第一次出现 000 是什么时候. 如果第一个 000 是 a2id+1a^{2^id} + 1a2id+1, 则 a2id≡n−1(modn)a^{2^id} \equiv n - 1 \pmod na2id≡n−1(modn). 那么将它平方后一定得到 111, 因此之后的项都是 1+1=21 + 1 = 21+1=2. 如果第一个 000 是 ad−1a^d - 1ad−1, 说明 ad=1a^d = 1ad=1, 因为 111 的正整数次方都是 111, 那么从第二项开始, 就一定都是 1+1=21 + 1 = 21+1=2.

接下来就是序列中没有 000 的情况.

讨论 an−1a^{n - 1}an−1 是否为 111. 如果 an−1a^{n - 1}an−1 为 111, 说明是在没有出现 −1-1−1 的情况下就平方出了 111, 根据前面推得的结论, 不是 111 或 −1-1−1 的数字平方得到 111, 则 nnn 是合数, 所以这时判为合数是正确的. 如果 an−1a^{n - 1}an−1 不是 111, 那么根据费马素性检验, nnn 是一个合数.

因此我们只要判断所有 ad−1a^d - 1ad−1, a2id+1a^{2^id} + 1a2id+1 是否全部非 000, 如果不存在 000 就认为 nnn 是个合数即可. 算法实现时先求 ada^dad, 可以快速得到前两项, 然后不断将 ada^dad 平方即可得到所有的项加 −1-1−1 的结果.
