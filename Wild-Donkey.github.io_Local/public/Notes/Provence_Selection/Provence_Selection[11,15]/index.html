Day 121212 Apr 15, 2022, Friday

还是那道题

看了题解发现我超脱了题解的路数, 以至于题解的方法无法优化这个做法, 而我也完全没有办法优化这个做法. 重新审视我们的转移, 发现每个阶段的转移只和当前列的状态有关, 而当前列为空白时, 所有状态从上一个阶段到下一个阶段的转移是相同的. 而当前列非空白时, 这个阶段有效状态数最多是 O(c)O(c)O(c) 的.

仍然是遇到第二行有确定颜色就交换两行, 我们用 fi,jf_{i, j}fi,j​ 表示第 iii 个非空列, 第二行是颜色 jjj 的方案数. 处理 gi,0/1/2/3/4g_{i, 0/1/2/3/4}gi,0/1/2/3/4​ 用来转移, 表示相邻两个非空列中间隔了 iii 列, 两端状态为 0/1/2/3/40/1/2/3/40/1/2/3/4 五种情况的转移系数. 两端共四个格子, 这五种情况分别是:

  • 000 表示四个颜色各不相同
  • 111 表示同行的颜色相同而同列的不同
  • 222 表示对角线上颜色相同而同行或同列颜色不同
  • 333 表示只有一行颜色相同, 其余两个格子为不同颜色
  • 444 表示只有一个对角线颜色相同, 其余两个格子为不同颜色.

发现有情况无法转移, 就是两端列存在空白列的情况, 这时需要预处理第一个阶段的 DP 值, 并且通过最后一个阶段的 DP 值算出答案.

ggg 数组线性递推即可求出.

发现瓶颈是对于每个阶段的 fff 数组进行全局修改查询和单点修改和查询, 由于模数较大, 所以无法线性, 但是可以通过线段树做到 O(nlog⁡c+c)O(n\log c + c)O(nlogc+c). 把快速查询的部分分代码复制过来, 删除全局赋值即可拿到 100′100'100′ 的好成绩.

Day 131313 Apr 16, 2022, Saturday

SDOI2019 热闹的聚会和尴尬的聚会

因为一个方案中, 第一天和第二天是互相独立的, 因此我们要分别使得 ppp 和 qqq 最大. 如果 ppp 和 qqq 分别最大的时候也不能满足条件, 则无解.

对于 ppp, 我们用单调队列从小到大维护每个点的度数, 每次取度最小的点删掉, 这个点的度数作为此时此刻还没删掉的点的集合的 ppp, 维护一个 ppp 的最大值, 和达到这个最大值时删掉的点集, 输出时可以将没删掉的点输出.

qqq 的最大值显然是求最大独立集, 这个问题是经典 NPC, 显然不能去求 qqq 的最大值, 只能求一个近似解.

一开始想的是, 随便找出一个极大的独立集, 据说这样能过. 方法是: 随机一个排列, 依次选择, 如果一个点是已经被选的邻居, 则跳过, 否则选择.

但是发现每次选一个点, 都会使它当前度数个点不能被选择, 所以感性地, 每次选择度数最小的点. 这样可以证明是一定有解的.

证明是这样的: 如果第 iii 次选择的点使 CgiCg_iCgi​ 个点从可以被选择变成了不能被选择, 一共选则了 qqq 个. 那么一定有 ∑i=1q(Cgi+1)=n\sum_{i = 1}^q (Cg_i + 1) = n∑i=1q​(Cgi​+1)=n.

我们知道, 如果第 iii 个被选择的点使 CgiCg_iCgi​ 个点从可以被选择变成了不能被选择, 那么这个点当时的度数一定大于等于 CgiCg_iCgi​, 而每个点被扩展的时候, 一定会更新 ppp, 也就是说 p≥Cgip \geq Cg_ip≥Cgi​.

我们需要满足的条件是 ⌊np+1⌋≤q\lfloor \frac{n}{p + 1}\rfloor \leq q⌊p+1n​⌋≤q, ⌊nq+1⌋≤p\lfloor \frac{n}{q + 1}\rfloor \leq p⌊q+1n​⌋≤p, 转化为 ⌊np+1⌋<q+1\lfloor \frac{n}{p + 1}\rfloor < q + 1⌊p+1n​⌋<q+1, ⌊nq+1⌋<p+1\lfloor \frac{n}{q + 1}\rfloor < p + 1⌊q+1n​⌋<p+1, 最后是 n<(q+1)(p+1)+n%(p+1)n < (q + 1)(p + 1) + n \% (p + 1)n<(q+1)(p+1)+n%(p+1), n<(q+1)(p+1)+n%(q+1)n < (q + 1)(p + 1) + n \% (q + 1)n<(q+1)(p+1)+n%(q+1). 一个更严格的界是 n<(q+1)(p+1)n < (q + 1)(p + 1)n<(q+1)(p+1), 拆括号可得 n<pq+p+q+1n < pq + p + q + 1n<pq+p+q+1.

因为 p≥Cgip \geq Cg_ip≥Cgi​, 所以 pq+q≥∑i=1q(Cgi+1)=npq + q \geq \sum_{i = 1}^q (Cg_i + 1) = npq+q≥∑i=1q​(Cgi​+1)=n, 因此 pq+q+p≥npq + q + p \geq npq+q+p≥n, pq+q+p+1>npq + q + p + 1 > npq+q+p+1>n.
